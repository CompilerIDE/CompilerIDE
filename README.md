## 拥有的功能

- [x] 代码编译运行与修改
- [x] 深浅主题切换
- [x] 自动括号补全
- [x] 自动缩进
- [x] 选择编译器路径或自动配置路径
- [x] 查找、替换
- [x] 项目管理器
- [x] 竞赛模式
- [x] 代码补全
- [x] 诊断工具、终端、代码片段
- [x] 洛谷题目爬取器
- [x] DeepSeek AI 助手
- [x] 对拍器、数据评测器
- [x] 代码格式化
- [x] 临时编译、临时编译运行
